Что выведет программа? Объяснить вывод программы.

```go
package main
 
import (
    "fmt"
)
 
func main() {
// a[5] {76, 77, 78, 79, 80} in 0
    a := [5]int{76, 77, 78, 79, 80}

// b[4] {77, 78, 79} in 1
    var b []int = a[1:4]

// out <- "[77 78 79]\n"
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]

```
